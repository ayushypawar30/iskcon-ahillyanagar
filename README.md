# 🛕 ISKCON Ahilyanagar Website

🌐 Live Website: https://iskconahilyanagar.org

A fully functional and responsive website developed for ISKCON Ahilyanagar. This project focuses on providing information about temple activities, events, दर्शन (Darshan), donations, and spiritual resources in a clean and user-friendly way.

---

## ✨ Features

- 🛕 Temple Information & दर्शन details  
- 📅 Events & Festival Updates  
- 💳 Donation System  
- 📸 Gallery Section  
- 📞 Contact & Location Details  
- 📱 Fully Responsive Design  
- ⚡ Fast and optimized performance  

---

## 🛠️ Tech Stack

### 👨‍💻 Backend
- Python (Flask)

### 🌐 Frontend
- HTML  
- CSS  
- JavaScript  

### 🗄️ Database
- MySQL
